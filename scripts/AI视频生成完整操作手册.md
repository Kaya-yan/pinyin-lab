## 四、平台操作步骤

### 4.1 即梦AI操作步骤（推荐）

#### 步骤1：登录
1. 打开浏览器，访问：https://jimeng.jianying.com
2. 点击右上角"登录"
3. 使用抖音/西瓜/头条账号扫码或手机号登录

#### 步骤2：进入视频生成
1. 登录后，点击顶部导航栏的"生成"
2. 在下拉菜单中选择"视频生成"
3. 进入视频生成页面

#### 步骤3：选择模式
1. 在页面中找到模式选择区域
2. 选择"**首尾帧**"模式（关键！）
3. 如果没有"首尾帧"，选择"图生视频"模式

#### 步骤4：上传图片
1. **上传首帧**：
   - 点击"首帧"或"开始帧"上传区域
   - 在文件选择器中，导航到 `C:\Users\ht\Documents\pinyinlab\pinyin-lab\public\frames\`
   - 选择对应的声母尾帧，如 `initial_b_end.jpg`
   - 等待上传完成

2. **上传尾帧**：
   - 点击"尾帧"或"结束帧"上传区域
   - 选择对应的韵母首帧，如 `final_a_start.jpg`
   - 等待上传完成

#### 步骤5：输入提示词
1. 在提示词输入框中，粘贴对应的提示词
2. 例如生成 "b → a" 的过渡：
   ```
   卡通风格口腔剖面图，舌头从双唇紧闭位置平滑移动到大口张开位置，
   保持TELINK教学视频风格，0.5秒，流畅自然，学术风格
   ```

#### 步骤6：设置参数
| 参数 | 设置值 | 说明 |
|------|--------|------|
| 时长 | 0.5秒 | 过渡视频时长 |
| 画质 | 高清 | 保证清晰度 |
| 比例 | 保持原图比例 | 与帧图片一致 |

#### 步骤7：生成视频
1. 点击"生成"或"开始生成"按钮
2. 等待生成完成（通常需要10-30秒）
3. 预览生成的视频

#### 步骤8：下载保存
1. 如果效果满意，点击"下载"按钮
2. 保存到本地，命名为：`transition_b_a.mp4`
3. 命名规则：`transition_{声母}_{韵母}.mp4`

## 五、批量生成策略

### 5.1 测试阶段（先测3-5个）

**推荐测试组合**：
1. `b → a`（简单，双唇到开口）
2. `q → i`（中等，舌面到前高）
3. `m → a`（简单，鼻音到开口）
4. `d → i`（中等，舌尖到前高）
5. `h → ou`（稍难，舌根到圆唇）

### 5.2 评估标准

每个测试视频检查：
- [ ] 首帧是否与上传的声母尾帧一致
- [ ] 尾帧是否与上传的韵母首帧一致
- [ ] 过渡是否平滑，没有跳跃感
- [ ] 舌头移动方向是否正确
- [ ] 风格是否与原有TELINK视频相似
- [ ] 时长是否合适（0.5秒）

### 5.3 批量生成优先级

**第一优先级（最常用组合）**：
```
ba, bi, bo, pa, pi, po, ma, mi, mo,
di, ding, ti, ting, ni, ning, li, ling,
ge, gong, ke, kong, he, hou,
ji, jia, qi, qia, xi, xia,
zhi, chi, shi, ri, zi, ci, si
```

**第二优先级（其他组合）**：
- 剩余的声母+韵母组合

## 六、质量检查标准

### 6.1 技术检查

| 检查项 | 合格标准 | 不合格处理 |
|--------|---------|-----------|
| 时长 | 0.5秒 ± 0.1秒 | 重新生成或剪辑 |
| 分辨率 | 与原始视频一致 | 重新生成 |
| 帧率 | 与原始视频一致 | 重新生成 |
| 首帧匹配 | 与声母尾帧一致 | 检查上传是否正确 |
| 尾帧匹配 | 与韵母首帧一致 | 检查上传是否正确 |

### 6.2 视觉检查

| 检查项 | 合格标准 | 不合格处理 |
|--------|---------|-----------|
| 过渡平滑 | 无明显跳跃或闪烁 | 调整提示词重新生成 |
| 舌头方向 | 符合发音学规律 | 修改提示词描述 |
| 风格一致 | 与TELINK视频风格相似 | 增加风格描述词 |
| 画面清晰 | 无模糊或变形 | 提高画质设置 |

### 6.3 语言学检查

| 检查项 | 合格标准 |
|--------|---------|
| 发音位置 | 声母结束位置正确 |
| 发音位置 | 韵母开始位置正确 |
| 移动路径 | 符合实际发音时的舌位变化 |
| 口型变化 | 符合实际发音时的口型变化 |

## 七、常见问题解决

### Q1: 生成的视频风格与原有视频不一致？

**解决方案**：
1. 在提示词中增加更多风格描述：
   ```
   卡通风格口腔剖面图，医学教学动画风格，
   线条简洁，色彩柔和，浅蓝色背景，
   保持TELINK教学视频风格...
   ```
2. 确保上传的帧图片清晰
3. 尝试不同的AI平台

### Q2: 过渡不自然，有跳跃感？

**解决方案**：
1. 增加过渡时长到1秒
2. 在提示词中增加：
   ```
   极其平滑的过渡，渐变过程，
   缓慢移动，无跳跃...
   ```
3. 检查首尾帧是否匹配（同一角度、同一风格）

### Q3: 舌头移动方向错误？

**解决方案**：
1. 在提示词中明确描述移动路径：
   ```
   舌头从【具体起点】沿【具体路径】移动到【具体终点】
   ```
2. 参考发音位置表，确保描述准确

### Q4: 生成失败或报错？

**解决方案**：
1. 检查图片格式是否为JPG
2. 检查图片大小是否合适（建议不超过2MB）
3. 检查网络连接
4. 尝试刷新页面或重新登录
5. 尝试更换浏览器

### Q5: 视频时长无法控制？

**解决方案**：
1. 有些平台不支持精确时长控制
2. 生成后使用剪辑软件调整时长
3. 或使用ffmpeg裁剪：
   ```bash
   ffmpeg -i input.mp4 -t 0.5 -c copy output.mp4
   ```

### Q6: 如何快速找到对应的帧文件？

**解决方案**：
1. 使用文件资源管理器的搜索功能
2. 搜索格式：`initial_{声母}_end.jpg` 或 `final_{韵母}_start.jpg`
3. 或查看 `frame-index.json` 文件
