- `README.md`
- `PRODUCT.md`

- `prompts/` — historical Claude / development prompts
- `reviews/` — project review and audit notes
- `research/` — competitor analysis and early proposal material

Files in this directory are preserved for reference only and should not be treated as the current definition of the product.
